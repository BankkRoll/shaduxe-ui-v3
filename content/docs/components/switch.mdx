## Props

| Prop         | Type                                                  | Description                           | Default   |
|--------------|-------------------------------------------------------|---------------------------------------|-----------|
| variant      | `default` \| `rounded` \| `square` \| `ios`            | Style variant of the switch           | `default` |
| size         | `icon` \| `xs` \| `sm` \| `md` \| `lg`                 | Size of the switch                    | `md`      |
| thumbVariant | `default` \| `rounded` \| `square` \| `ios`            | Style variant of the switch thumb     | -         |
| showLabels   | `boolean`                                              | Show text labels                      | `false`   |
| labelOn      | `string`                                               | Text for the "on" label               | `"On"`    |
| labelOff     | `string`                                               | Text for the "off" label              | `"Off"`   |
| showIcons    | `boolean`                                              | Show icons in the switch thumb        | `false`   |
| className    | `string`                                               | Additional CSS classes                | -         |

### Supported HTML Attributes

Because the Switch uses Radix UI's `Switch` component, it inherits and supports **all standard HTML input attributes**, including:

- `checked`
- `defaultChecked`
- `required`
- `name`
- `value`
- `onCheckedChange`
- `disabled`
- `aria-*` attributes
- `data-*` attributes
- Event handlers like `onFocus`, `onBlur`, etc.

The Switch component uses Radix UI's primitives, ensuring proper accessibility, keyboard navigation and semantic HTML.
