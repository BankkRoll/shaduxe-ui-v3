## Usage

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar"
```

```tsx
<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" />
  <AvatarFallback>CN</AvatarFallback>
</Avatar>
```

## Props

| Prop     | Type                | Description                                | Default   |
|----------|---------------------|--------------------------------------------|-----------| 
| variant  | `circle` \| `square` \| `rounded` | Shape of the avatar | `circle`  |
| size     | `xs` \| `sm` \| `md` \| `lg` \| `xl` \| `2xl` | Size of the avatar | `md` |
| className| `string`            | Additional CSS classes                     | -         |

### Supported HTML Attributes

Because the Button uses Radix UI, it inherits and supports **all standard HTML image attributes**, including:

- `src`
- `alt`
- `width`
- `height`
- `aria-*` attributes
- `data-*` attributes
- Event handlers like `onError`, `onLoad`, etc.

The Avatar component uses Radix UI's primitives, ensuring proper accessibility, keyboard navigation and semantic HTML.
