## Props

| Prop         | Type                                                     | Description                                       | Default    |
|--------------|----------------------------------------------------------|---------------------------------------------------|------------|
| variant      | `default` \| `destructive` \| `outline` \| `secondary` \| `ghost` \| `link` \| `expandIcon` \| `ringHover` \| `ringHoverOutline` \| `shine` \| `gooeyRight` \| `gooeyLeft` \| `linkHover1` \| `linkHover2` | Button style variant                  | `default`  |
| size         | `icon` \| `xs` \| `sm` \| `md` \| `lg` \| `xl` \| `2xl`   | Size of the button                                | `md`       |
| asChild      | `boolean`                                                | Render as a child component                       | `false`    |
| Icon         | `React.ElementType`                                      | Icon component to display                         | -          |
| iconPlacement| `left` \| `right`                                        | Position of the icon relative to the text         | -          |
| className    | `string`                                                 | Additional CSS classes                            | -          |

### Supported HTML Attributes

Because the Button uses Radix UI, it inherits and supports **all standard HTML button attributes**, including:

- `onClick`
- `disabled`
- `type` (`"button"`, `"submit"`, `"reset"`)
- `form`
- `formAction`
- `name`
- `value`
- `aria-*` attributes
- `data-*` attributes
- Event handlers like `onMouseEnter`, `onFocus`, etc.

The Button component uses Radix UI's primitives, ensuring proper accessibility, keyboard navigation and semantic HTML.
